FAQ

Questions

The short version: WhisprCatch is the open-source Wispr Flow alternative for Linux — same hold-to-talk dictation, none of the cloud.

Is there a Wispr Flow alternative for Linux?

Yes — that's exactly what WhisprCatch is. Wispr Flow (often searched as "Whisper Flow") doesn't ship a Linux app; WhisprCatch gives you the same flow on Linux: hold a key, speak, release, and punctuated text is typed at your cursor in any app. Free and open source, with no account and no subscription.

Is it really free?

Completely. $0 forever, no account, no trial, no subscription. Every line is MIT-licensed on GitHub — read it, audit it, fork it.

Does my voice leave my laptop?

No. Speech-to-text runs on your CPU with zero network calls while you dictate — fully offline after a one-time model download. Your audio is decoded in memory and never uploaded.

Which Linux distros does it work on?

Ubuntu and Debian (x86-64) via the .deb today; anything else builds from source. It types into whatever has focus — editors, browsers, chat apps, terminals.

What about macOS?

An Apple Silicon build is in the works — join the waitlist above and you'll get exactly one email when it ships.
